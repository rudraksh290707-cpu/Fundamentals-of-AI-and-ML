# smart-decision-engine (C - CLI Project)
A smart decision-making CLI application in C that simulates real-life choices using structured evaluation, weighted scoring, and logical reasoning.

##  Overview
Life Decision Simulator is a command-line based application written in C that helps users make structured and logical decisions. It evaluates multiple options based on user-defined factors and their importance (weights), and suggests the best possible choice.

This project demonstrates how programming logic can be applied to real-life decision-making scenarios.

##  Objectives
- To help users make better decisions using a structured approach
- To implement a weighted scoring system
- To build a fully executable CLI-based application in C

##  Features
- Fully command-line based (no GUI required)
- Supports multiple options and decision factors
- Weighted scoring mechanism for evaluation
- Displays the best decision clearly
- Modular code using multiple source and header files

##  Tech Stack
- Programming Language: C
- Compiler: GCC (or any standard C compiler)
